Downloading dio

dio is now available in standard Solaris package format. Use the pkgadd command to install the package. This is the download format I would recommend as the package includes a proper man page as well. The package installs into /usr/local.

If you are having problems downloading from here (using HTTP) then you can download using FTP from Sunfreeware. Also, if would like dio for a version of Solaris that is not available here, then try sunfreeware.com
